Erlang C Visualizer - Embed Examples

This page demonstrates different ways to embed the Erlang C Visualizer in your articles or websites.

Example 1: Full Application

Embed the complete application with all features:

<iframe src="https://YOUR_USERNAME.github.io/erlang-c" width="100%" height="900px" frameborder="0"></iframe>

Example 2: Pre-configured Individual Server

Pre-set parameters for a specific scenario:

<iframe src="https://YOUR_USERNAME.github.io/erlang-c?tab=individual&arrivalRate=150&serviceTime=50&workers=8" width="100%" height="800px" frameborder="0"></iframe>

Example 3: Fleet Optimization

Show the fleet optimization tab with specific configuration:

<iframe src="https://YOUR_USERNAME.github.io/erlang-c?tab=fleet&arrivalRate=200&servers=5&workers=6" width="100%" height="900px" frameborder="0"></iframe>

Example 4: Responsive Embed

Responsive embed that adapts to container width:

<div style="position: relative; padding-bottom: 75%; height: 0; overflow: hidden;"> <iframe src="https://YOUR_USERNAME.github.io/erlang-c" style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; border: 0;" frameborder="0"> </iframe> </div>
